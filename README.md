# Create SFTP server with Multiple Jailed Users
# How to run this
```
sudo bash create_sftp_users.sh
```
# Script Execution Flow
```
Script will  ask you to enter your sftp username:
   -  if you press Return without entering any name script will be exit.
   - Otherwise a sftp username will be created with a name whatever you will enter.
Script will ask you to enter your sftp password:
  -  if you press Return without entering any password script will generate random password automatically
  -  Otherwise  a password whatever you will enter
Script will ask you to enter your sftp HomePath:
  -  if you press Return without entering any HomePath will be default
  -  Otherwise  a HomePath whatever you will enter
```
